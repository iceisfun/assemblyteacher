# Binary and Hexadecimal

The first lesson. Establishes base-2 and base-16 notation, explains why hex is
universal in systems work (one digit = four bits, exactly), and plants the idea
that a byte's meaning comes from its interpreter rather than from the byte.

- `lesson.md` — the lesson, with its exercises in the front matter.
- `examples/bases.asm` — the same value written in three bases, assembling to
  identical bytes. Assembled by the test suite.

Nothing here depends on any other lesson.
